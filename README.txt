To run the game, open the "Green Eyes Invasion" folder from console and input "npm run dev" tp test.

How to manage the levels:
The game comes with a DEMO level, and 3 levels that can be loaded with in the game (01, 02, 03). To add more levels, run the level editor to create your own levels, then copy the jSON file created on the "Levels" folder from the level editor and paste it in the "levels" folder of the game, and then it will be possible to load from the game.
