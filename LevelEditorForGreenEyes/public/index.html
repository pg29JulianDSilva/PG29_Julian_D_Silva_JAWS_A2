Tag references for the levels.

cannon:

Makes reference from where the player will start launchg the Animals. (Recomended to place only one).

enemy:

Makes reference from where the enemies will be located (to be destroyed by the player).

block(H and V):

Block to construct the enemy structure, to protect it. This elements ARE DESTRUCTABLE by the player.

TNT:

Small block that, if collides, will destroy all DESTRUCTABLE elements in radius, and itself.

Terrain:

This are also bulding blocks, however, this ones ARE NOT DESTRUCTABLE by the player. Use for floor and similar structures.
